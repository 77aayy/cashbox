# صندوق التقفيل — نسخة جديدة

تطبيق تقفيل صندوق محلي بالكامل (بدون سيرفر أو Firebase). كل البيانات في المتصفح (localStorage).

## التشغيل

```bash
cd cashbox-new
npm install
npm run dev
```

ثم افتح http://localhost:5173

## ما يوفره التطبيق

- **دخول**: اسم الموظف فقط (حروف بدون أرقام).
- **جدول تقفيل**: موظف، وقت (ميلادي مع اسم اليوم)، كاش، مدى، فيزا، تحويل بنكي، رصيد البرنامج كاش، اجمالى الموازنه، انحراف، ملاحظات، حالة.
- **صف أول تلقائي** عند فتح التطبيق إن لم يكن هناك بيانات.
- **إغلاق الشفت**: زر في كل صف نشط → عد تنازلي 10 ثوانٍ مع زر «تراجع». بعد 10 ثوانٍ يُغلق الصف ويُضاف صف جديد.
- **لا إضافة صف جديد** ما دام هناك صف نشط أو صف في مهلة التراجع.
- **عرض 5 صفوف** ثم زر «عرض مزيد».
- **التواريخ ميلادية** بصيغة مثل: الثلاثاء 2 يناير 10:52 م.
- **الصف المغلق** غير قابل للتعديل.
