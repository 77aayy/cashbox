# صندوق التقفيل (CashBox)

المشروع موحّد في **مجلد واحد** — جذر المستودع.

## التشغيل المحلي

- **طريقة سريعة:** شغّل `start-cashbox.bat` (يفتح السيرفر والمتصفح).
- **أو من الطرفية:**
  ```bash
  npm run dev
  ```
  ثم افتح: <http://localhost:5173/cashbox/>

## البناء والنشر على GitHub Pages

```bash
npm run build
npm run deploy
```

بعدها الموقع يتحدّث على: <https://77aayy.github.io/cashbox/>

## ملاحظة عن المجلد القديم

إذا ظهر مجلد `cashbox` (بداخله نسخة قديمة) وأردت حذفه: أغلق Cursor وأي نافذة طرفية تستخدم المشروع، ثم احذف مجلد `cashbox` يدوياً. المشروع يعمل الآن من الجذر فقط.
